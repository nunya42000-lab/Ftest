📖 NEXUS PRIME v40.20 - THE MASTER CODEX
SOVEREIGN ARCHITECTURE
Nexus Prime is an AST-aware, mobile-native Integrated Development Environment.
TIER I: THE KINETIC INTERFACE
1. The Infinite Ribbon
The top navigation bar is a seamless loop. Swipe horizontally to access secondary tools (Vault, Radar, Settings). It will wrap around infinitely.
2. Kinetic Widgets (Util / Search / Writer)
Widgets are floating modules designed for one-handed operation.
  • Draggable: Grab handles (⋯) to reposition anywhere.
  • Magnetic Docking: Double-tap a handle to snap the widget to the screen edge.
  • Visibility: Toggle widgets via the 🧩 WIDGETS dropdown in the ribbon.
3. Edit-Lock Protocol
Tap the 🔒 READ ONLY icon to cycle states:
  • READ ONLY: Safe mode. No inputs possible.
  • UTIL MODE: Interaction enabled, but OS keyboard suppressed. Use for coding with Widgets/Keyboard Monolith.
  • FULL EDIT: Standard input mode. OS keyboard will activate on tap.
TIER II: VORTEX FILE SYSTEM (VFS)
1. VFS Explorer
Files are stored in the Sovereign Database (Dexie.js). They persist even if you close the browser. Open the 📁 VORTEX panel to manage sectors.
2. Matrix Build Operations
Located in the Vortex panel, these are high-level build commands:
  • ⚡ BUNDLE: Compiles the entire project (HTML+CSS+JS) into a single standalone HTML file. Escapes internal tags to prevent breakage.
  • ⚡ MERGE JS: Concatenates all .js files into a single production script.
  • ✂️ SPLIT JS: Uses the AST to identify independent functions/classes and shatters a single file into multiple modular .js sectors.
  • 🔗 COMBINE: Merges specific files and automatically re-routes import and src paths throughout the rest of the project.
TIER III: NEURAL ENGINES (DIAGNOSTICS)
1. Sentinel AST Medic
Sentinel performs 16+ structural checks. It detects Infinite Loops, XSS Vulnerabilities, Recursion Risks, and Zombies (unused variables).
Action: Click FIX ALL to apply bulk mutations to your code automatically.
2. Neural Heuristics (Intel)
A high-level analytical scan. It calculates "Code Health" based on payload size, security risks, and structural patterns.
Action: Tap the large % Health number in the Intel modal to view Deduction Logs.
3. Radar Graph
A visual dependency map. It tracks how files are connected.
  • Green Tree: Fully linked and functional paths.
  • Red Orphans: Files that are not imported by anything (Dead Code).
TIER IV: TEMPORAL MECHANICS
1. The Commander Deck (Merge Engine)
When performing complex merges or restoring snapshots, use the Merge Engine. It provides a line-by-line diff. Select Left, Right, or Both for every conflict to assemble the final code.
2. Sovereign Vault
A persistent snippet library. Highlighting code and selecting + GRAB stores the logic in your Vault. Vault items are available across all files and sessions.
TIER V: CMD & SHELL
1. Native VFS Commands
Type the following into the CMD slider for fast file manipulation:
  • ls - List all project sectors.
  • touch [filename] - Create a new file instantly.
  • rm [filename] - Delete a file.
  • cat [filename] - View file content in terminal output.
  • clear - Wipe current editor content.
TIER VI: OMNI GESTURE LAYER
3-Finger Gesture Activation
Touch the screen with 3 fingers to activate the Omni Layer. Draw a path to trigger commands:
  • D → L : Fire Sentinel Deep Lint.
  • U → D → U → D : Run Neural Heuristics (Intel).
  • D → R → U → L : Auto-Format active file.
